** 30. __ hashing uses a secondary hash function h' on the keys to determine the increments, aiming to avoid the problem.
*** sol:
Double hashing（双重哈希）—— 使用第二个哈希函数 h'(key) 来决定探测步长，每次不是固定步长（如+1），而是 (index + i * h'(key)) % table.length，这样可以避免一次聚集（primary clustering）的问题。

** Hash Table Collision
- A hash table stores data in an array.
- A hash function maps a key to an index.
- Collision（冲突）happens when two different keys are mapped to the same index.
*** Collision Resolution
- Separate chaining（分离链表法）
  - Each slot stores a linked list.
  - All elements mapped to the same slot are stored in that linked list.
- Open addressing（开放寻址法）
  - All elements are stored inside the hash table array.
  - If a slot is occupied, the algorithm probes another slot.
*** Double Hashing
- Double hashing is a type of open addressing.
- It uses a second hash function h'(key) to decide the probing step size.
- Formula:
  index_i = (h(key) + i * h'(key)) mod table.length
- Purpose:
  reduce primary clustering（一坨连续位置被占用的问题）.

        for (String book : fictionQueue) {
            bookCounts.putIfAbsent(book, new ArrayList<>());
            bookCounts.get(book).add(1);

        for (String book : historicalQueue) {
            bookCounts.putIfAbsent(book, new ArrayList<>());
            bookCounts.get(book).add(2);

            int flag1, flag2, flag3 = 0;
            flag1 = flag2 = flag3 = 0;

import java.util.Arrays;
import java.util.Comparator;

public class Main {
    // Task1: Use the follwing insertionSort() method with the Comparable interface
    // to sort the array of employee IDs in ascending order.

    public static <E extends Comparable<E>> void insertionSort(E[] array) {
        for (int i = 1; i < array.length; i++) {
            E key = array[i];
            int j = i - 1;
            while (j >= 0 && array[j].compareTo(key) > 0) {
                array[j + 1] = array[j];
                j--;
            }
            array[j + 1] = key;
        }
    }

    // Task2: Use the following insertionSort() method with the Comparator interface
    // to srot the array of employee names in a case-insensitive manner
    public static <E> void insertionSort(E[] list, Comparator<? super E> comparator) {
        for (int i = 1; i < list.length; i++) {
            E key = list[i];
            int j = i - 1;
            while (j >= 0 && comparator.compare(list[j], key) > 0) {
                list[j + 1] = list[j];
                j--;
            }
            list[j + 1] = key;
        }
    }

    public static void main(String[] args) {
        Integer[] employeeIds = { 102, 101, 104, 103 };
        String[] employeeNames = { "John", "Alice", "Bob", "Charlie" };

        insertionSort(employeeIds);
        System.out.println("Sorted Employee IDs: ");
        System.out.println(Arrays.toString(employeeIds));

        insertionSort(employeeNames, String.CASE_INSENSITIVE_ORDER);
        System.out.println("Sorted Employee Names: ");
        System.out.println(Arrays.toString(employeeNames));
    }
}
