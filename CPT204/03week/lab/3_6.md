# 练习 3.6：抽象类 vs. 接口

1. 看下面代码能不能成功编译；如果不能，原因是什么？

```java
public interface Animal {
    String name;   // 表示动物名称的数据字段
    void makeSound();   // 发出动物叫声的抽象方法
}
```

- sol:
  interface can not have field

2. 看下面代码能不能成功编译；如果不能，原因是什么？

```java
public interface MyInterface2 {
    void method1();   // 没有实现的抽象方法
    void method2();   // 没有实现的抽象方法
    void method3() {
        // 带有实现的具体方法
        System.out.println("Method 3 implementation");
    }
}
```

- sol
  interface can not have impled method

3. 下面代码的问题是什么？应该如何解决？

```java
public abstract class MyClass {
    public MyClass() {
        System.out.println("Abstract class constructor");
    }

    public static void main(String[] args) {
        MyClass obj = new MyClass();
    }
}
```

- sol
  abstract class can not have main entrance
