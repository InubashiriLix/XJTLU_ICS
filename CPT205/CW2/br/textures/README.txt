This folder should contain 24-bit uncompressed BMP textures used by the program.

Required filenames:
- concrete.bmp
- concrete_dark.bmp
- windows_blue.bmp
- windows_warm.bmp
- road.bmp
- metal_junk.bmp
- neon1.bmp
- neon2.bmp
- ocean.bmp
- sky_night.bmp
- rust.bmp

You can paint them yourself using any simple drawing tool.
