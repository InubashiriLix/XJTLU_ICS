## 🚀 快速开始

```bash
# 1. 克隆仓库
git clone https://github.com/your-username/your-repo.git
cd your-repo

# 2. 下载neovim以及Lazyvim，并且安装neorg插件
# 我是推荐使用nvim的，但是你也可以使用别的编辑器，如果没有neorg插件或者nerog支持的编辑器，那阅读笔记可能比较痛苦

# 代码部分需要的环境：
#  python
#     sklearn, matplotlib, pandas, numpy
#  dotnet: 
#     dotnet sdk 8+
#  cpp:
#     cpp 17+
# java:
#     jdk 17+
```
