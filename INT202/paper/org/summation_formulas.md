# 常见累加公式汇总

---

## 一、基本累加公式

| 编号 | 累加式 | 闭式公式 | 复杂度 |
|------|--------|----------|--------|
| 1 | `∑(i=1 to n) 1` | `n` | Θ(n) |
| 2 | `∑(i=1 to n) i` | `n(n+1)/2` | Θ(n²) |
| 3 | `∑(i=1 to n) i²` | `n(n+1)(2n+1)/6` | Θ(n³) |
| 4 | `∑(i=1 to n) i³` | `[n(n+1)/2]²` | Θ(n⁴) |
| 5 | `∑(i=1 to n) iᵏ` | — | Θ(nᵏ⁺¹) |
| 6 | `∑(i=0 to n) aⁱ`（等比） | `(aⁿ⁺¹−1)/(a−1)`, a≠1 | Θ(aⁿ) |
| 7 | `∑(i=1 to n) 2ⁱ` | `2ⁿ⁺¹−2` | Θ(2ⁿ) |
| 8 | `∑(i=1 to n) log i` | `n log n − n + O(log n)` | Θ(n log n) |

## 二、乘法法则

| 规则 | 说明 |
|------|------|
| `∑ c·f(i) = c·∑ f(i)` | 常数可提出 |
| `∑ [f(i) + g(i)] = ∑ f(i) + ∑ g(i)` | 加减可拆分 |

**例：** `∑(i=1 to n) (3i² + 2i) = 3·∑i² + 2·∑i = 3·Θ(n³) + 2·Θ(n²) = Θ(n³)`

---

## 三、嵌套累加（双重/多重求和）

| 累加式 | 等价形式 | 复杂度 |
|--------|----------|--------|
| `∑(i=1 to n) ∑(j=1 to n) 1` | `n²` | Θ(n²) |
| `∑(i=1 to n) ∑(j=1 to i) 1` | `∑ᵢ i = n(n+1)/2` | Θ(n²) |
| `∑(i=1 to n) ∑(j=i to n) 1` | 同上 | Θ(n²) |
| `∑(i=1 to n) ∑(j=1 to i) j` | `∑ᵢ i(i+1)/2` | Θ(n³) |
| `∑(i=1 to n) ∑(j=1 to i) i` | `∑ᵢ i²` | Θ(n³) |

## 四、常见叠加模式（对应代码循环）

### 模式 1：等比/二重循环
```
for i = 1 to n
    for j = 1 to n
        ... O(1)
```
时间复杂度：`∑(i=1 to n) ∑(j=1 to n) 1 = n²` → **Θ(n²)**

### 模式 2：三角形循环（j 依赖 i）
```
for i = 1 to n
    for j = 1 to i
        ... O(1)
```
时间复杂度：`∑(i=1 to n) i = n(n+1)/2` → **Θ(n²)**

### 模式 3：指数衰减
```
for i = 1 to n
    for j = 1 to 2^i
        ... O(1)
```
时间复杂度：`∑(i=1 to n) 2ⁱ = 2ⁿ⁺¹−2` → **Θ(2ⁿ)**

### 模式 4：对数型
```
for i = 1 to n
    j = i
    while j > 0
        j = j / 2
```
时间复杂度：`∑(i=1 to n) log i` → **Θ(n log n)**

## 五、常用技巧

| 技巧 | 用法 |
|------|------|
| 近似积分 | `∑(i=1 to n) f(i) ≈ ∫₁ⁿ f(x)dx`，用于估计 |
| 主项提取 | 只看闭式公式中最高次项，决定 Θ 界 |
| 裂项相消 | 如 `∑ 1/[i(i+1)] = 1−1/(n+1)` |

---

## 核心规律

> 对 `iᵏ` 求和，结果就是 **Θ(nᵏ⁺¹)**，这是判断复杂度的最快方法。
