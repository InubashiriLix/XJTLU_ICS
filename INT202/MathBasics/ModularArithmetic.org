* Modular Arithmetic
The division relationship (a = q * n + r) discussed in the previous section has two inputs (a and n) and two outpus (q and r). In modular arithmetic, we are interested in only one of the outputs, the remaineder r;
like a mod n = r (r is non-negative)
** Set of Residues
The modulo operation creates a set, which in modular arithmetic is referred to as the set of least residues modulo n or Zn
Zn = {0, 1, 2, 3, ... , n - 1}
for example Z5 = {0, 1, 2, 3, 4}
** Congruence
To show that two integers are congruent, we use the congruence operator (===): a===b(mod m) 
a is congruent to b modulo m 
for example: 17 === 5 (mod 12), 13 === 1 (mod 12)
-8 === 2 === 12 === 22 (mod 10)

a residue class [a] or [a]n is the set of integers congruent modulo n:
[a] = {..., a - 3n, a - 2n, a - n, a, a + n, a + 2n, ...}
for example:
[0] = {..., -15, -10, 5, 0, 5, 10, 15, ...}
[1] = {..., -14, -9, -4, 1, 6, 11, 16, ...}

** Inverse
*** Additive Inverse:
in Zn, two nuimbner are the additive inverse of each other if thier sum is congruent to 0 modulo n, which means
a + b === 0 (mod n)
- cuase 0 mod n = 0, there fore a + b mod n =0, which means
  that is a + b can be mod by n
- for example: in Z5 = {0, 1, 2, 3, 4}, we find the inverse of 2, which is 3, because 2 + 3 = 5, and 5 === 0 mod (5)
*** Multiplicative Inverse:
in Zn, two numbers are multiplicative inverse of each other if their product is congruent to 1 modulo n,
a * b === 1 (mod n)
- demote the multiplicative inverse of an element b by b^-1 (mod n)
- cuase 1 mod n = 1, there fore a * b mod n = 1, which means
  a * b can is larger than n by 1
- the extended euclidena algorithm find the multiplicative inversers of b in Zn when n and b are given and gcd(n, b) = 1
- noted that in modular arithemtic, an integer may or may not have a multiplicative inverse, when it does, the product of the integer and its multiplicative inverse is congruent to 1 modulo n
for example, in Z5 = { 0, 1, 2, 3, 4}, we find the multiplicative inverse of 2, which is 3, cause' 2 * 3 = 6 and 6 === 1 (mod 5)

** Different Sets
- Zn* is the set of all integers in Zn that fullfill that gcd(a, n) = 1
  Zn* = {[a]n ∈ Zn | gcd(a, n) = 1}
  for example: Z6 = {0, 1, 2, 3, 4, 5}, Z6* = {1, 5}
  when the modulus n is a prime number, then Zn* = Zn - {0}
