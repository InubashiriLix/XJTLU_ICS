* Show that 3-SAT belongs to the class NP;
   - intro:
     a bool formula like (x1 v not x2 v x3) ^ (not x1 v x4)
     - literal: the variable or its negation, for example: x1, not x2, x3, not x1, x4
     - clause: a disjunction of literals, for example: (x1 v not x2 v x3) and (not x1 v x4)
     - CNF: a bunch of clauses connected by conjunction, for example: (x1 v not x2 v x3) ^ (not x1 v x4)
     - 3-SAT: each clause has 3 literal SAT question
- solution:
  1. prove 3-SAT is in NP
     we can verify a proposed solution (certificate) in polynomial time, that is, we can check if the proposed solution satisfies the formula in polynomial time, because we only need to check each clause once, and each clause has at most 3 literals, so the time complexity is O(n), where n is the number of clauses.
     for example x1 = T, x2 = F, x3 =T
     then (x1 v not x2 v x3) = T

     we can write this: 
     Given a truth assignment as a certificate, we can verify whether each clause contains at least one true literal. Since there are polynomially many clauses and each clause has exactly three literals, the verification can be done in polynomial time. Therefore, 3-SAT belongs to NP.
